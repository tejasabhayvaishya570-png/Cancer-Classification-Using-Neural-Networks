# Cancer Classification Using Neural Networks

## Overview
This project predicts whether a breast tumor is Benign or Malignant using a Neural Network built with TensorFlow and Keras.

## Technologies Used
- Python
- TensorFlow
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab

## Dataset
Breast Cancer Wisconsin Dataset

## Features
- Data preprocessing
- Feature scaling
- Neural Network model
- Model evaluation
- Cancer prediction

## Results
- Training Accuracy: ~98%
- Test Accuracy: ~95–99%

## How to Run
1. Open the notebook in Google Colab.
2. Run all cells from top to bottom.
3. The model will train and classify tumors as Benign or Malignant.
